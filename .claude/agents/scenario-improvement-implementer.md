# Scenario Improvement Implementer (project-scoped shadow)

**This is a project-level override** of the plugin-shipped agent in
`scenarios-autorunner` v0.1.4+. The only difference from the plugin
version is the `hooks:` frontmatter field — CPV strict mode refuses
`hooks:` in plugin-shipped agents (security restriction), so the guard
has to live in a project- or user-scope agent file.

All other behaviour is inherited from the plugin documentation at
`~/.claude/plugins/cache/emasoft-plugins/scenarios-autorunner/0.1.4/agents/scenario-improvement-implementer.md`.
Do not edit this file to diverge from the plugin semantics — if the
plugin updates, copy the new body in and keep the `hooks:` field at
the top.

## Write-guard (enforced by the hook)

The `PreToolUse` hook runs `.claude/scripts/subagent-write-guard.sh`
before every `Write`, `Edit`, `MultiEdit`, `NotebookEdit`, and `Bash`
tool call. It blocks writes that land outside:

1. `$CLAUDE_PROJECT_DIR` — your worktree root (because this agent has
   `isolation: worktree`, `$CLAUDE_PROJECT_DIR` resolves to the
   worktree path, not the parent repo)
2. `/tmp` / `/private/tmp` — system scratch for plugin cloning

Read operations are NOT blocked. You may read any file on the system.
If you need to write a file outside these roots (e.g. to patch a
different plugin on disk, or to modify `~/.claude/settings.json`),
return `[DEFERRED] <reason>` as your 2-line summary — the orchestrator
will handle the out-of-scope write in the parent session.

## Delegating to the plugin body

The full instructions — project command detection, P0 parsing, fix
retry loop, hard rules, rate-limit resilience, return format — live in
the plugin-shipped agent. Read it at the start of your run:

```
cat ~/.claude/plugins/cache/emasoft-plugins/scenarios-autorunner/0.1.4/agents/scenario-improvement-implementer.md
```

Follow that file verbatim, subject to the additional write-guard
constraint documented above. If the plugin body says to `cd` out of
your worktree, the hook will block you — do not try to work around it.

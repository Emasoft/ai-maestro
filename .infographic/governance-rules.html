4 — SECURITY & NETWORK
MESH VPN

TAILSCALE VPN — PRIMARY SECURITY LAYER

All inter-agent communications, remote terminal access, cross-host governance, and mobile dashboard connections are secured through Tailscale's WireGuard-based mesh VPN. No traffic leaves the encrypted mesh.

WireGuard encryption (always-on)
Cross-host agent communication
Mobile/remote terminal access
Zero-config mesh networking
MagicDNS for agent discovery
Node identity via Tailscale keys
No exposed public ports
ACL-based access control

Authentication

  • Tailscale VPN as primary perimeter
  • Bearer token auth for agent-to-agent API calls
  • API keys: 256-bit random, SHA-256 hashed at rest
  • Constant-time key validation (timing-safe)
  • MANAGER + COS restricted to Claude clients

Rate Limiting

  • All password-protected endpoints rate-limited
  • Atomic check-and-record (no TOCTOU race)
  • Per-agent rate limit keys
  • Reset on successful auth
  • AMP route: max requests/min per agent

Cryptography

  • Ed25519 host keypairs (0o600 file perms)
  • Signed attestations with 5-min timestamp freshness
  • Recipient binding prevents cross-host replay
  • AMP messages signed with per-agent Ed25519 keys
  • bcrypt for governance passwords (72-byte limit)

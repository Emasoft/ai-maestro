### FIXED: Issue 1 — `yarn install` error handling in `act3_clone_and_build` (fresh install path)
- **Location**: Line 1018 (fresh install path inside `act3_clone_and_build`)
- **Before**: `yarn install || maestro_warn "yarn install had errors — continuing"`
- **After**: `yarn install || { maestro_fail "Failed to install dependencies."; exit 1; }`
- **Rationale**: On a fresh install, a failed `yarn install` leaves the application unbuildable. The script must exit immediately with a clear error rather than silently continue into a broken state. The update path at line 962 retains `maestro_warn` as specified — warn-and-continue is acceptable for updates where a partial dependency refresh may still work.
